\subsection{MDS embedding from relational distances}
  \label{subsec:mds-embedding-from-relational-distances}

  From the distance matrix $d_{ij}$, define the centered Gram matrix
  \begin{equation}
    G_{ij}
    =
    -\frac{1}{2}
    \big(d_{ij}^2 - d_{i\cdot}^2 - d_{\cdot j}^2 + d_{\cdot\cdot}^2\big),
    \label{eq:gram-matrix}
  \end{equation}
  with $d_{i\cdot}^2=\frac{1}{N}\sum_k d_{ik}^2$ and
  $d_{\cdot\cdot}^2=\frac{1}{N^2}\sum_{k\ell} d_{k\ell}^2$.

  Diagonalizing $G$ gives eigenpairs $(\lambda_k, v_k)$.
  The embedding in $\mathbb{R}^d$ is
  \begin{equation}
    x_i^{(a)} = \sqrt{\lambda_a}\,(v_a)_i,
    \qquad a=1,\dots,d,
    \label{eq:mds-embedding}
  \end{equation}
  yielding $d_{ij}\approx \|x_i-x_j\|$ in the projectable regime.
  The coordinates are defined up to global isometries.
